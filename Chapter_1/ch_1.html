Chapter 1 - Introduction

Java is an object-oriented programming language1 that is heavily utilized in back-end software development. Developers use this language to create applications on computers.

Example of Java Syntax

System.out.println("Hello, World!);

Creating your first program in JAVA

Use the following link to go to an online compiler: Repl.it

2. Create an account or log in with Google, GitHub, or Facebook

3. Click on the "+" icon and select "Java" as the repl language

4. Once you have created the repl, navigate to the file called "Main.java"

5. Control + A and delete everything on the left-hand side

Creating your first method - Addition

Main Method

class Main {
  public static void main( String [] args ) {
    //Printing Hello World!
    System.out.println("Hello World!");
  }
  public int sum( int a, int b ) {
    return a + b;
  }
}

What is a method header?

There are a few keywords within the highlight code that you need to understand while coding in Java.
"int" represents the return_type of the method. "sum" is the name of the method.
"return" declares what will be given to caller once the method is called upon.

How can we invoke this method?

Within the main method, call the function you created. In this case, you can call sum and give it two integers.

System.out.println( sum( 10, 20) );
